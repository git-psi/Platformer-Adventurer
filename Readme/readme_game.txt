*Game

This game is a platformer.
You will embody an adventurer...

The game is simple,
here's how to play it:
 -Arrow Right : Move to right
 -Arrow Left : Move to left
 -Backspace : Jump
 -Entry : Attack

Warning !
Every three sword strokes you have to wait
for the adventurer to rest.

You can pick up objects on the ground, they will help you.


Good adventure !
